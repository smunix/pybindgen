Class OverloadedWrapper

source code

An object that aggregates a set of wrapper objects; it generates a single python wrapper wrapper that supports overloading, i.e. tries to parse parameters according to each individual Function parameter list, and uses the first wrapper that doesn't generate parameter parsing error.
