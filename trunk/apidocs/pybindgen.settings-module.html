Variables Details

unblock_threads

Generate code to support threads. When True, by default methods/functions/constructors will unblock threads around the funcion call, i.e. allows other Python threads to run during the call.

Value:
False

error_handler

Custom error handling. Error handler, or None. When it is None, code generation exceptions propagate to the caller. Else it can be a pybindgen.utils.ErrorHandler subclass instance that handles the error.

Value:
None
